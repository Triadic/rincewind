Class Details



[line 50]

The FileRetriever actually fetches a files, and returns the File object.

There is a static File::create() class which actually instantiates the  FileRetriever and gets the file.  The FileRetriever is a class on it's own, so it can be passed to other  objects, so they don't have to call the static File:: functions (which  makes them rather difficult to test.)
