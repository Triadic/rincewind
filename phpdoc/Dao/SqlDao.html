method find [line 140]

Record find( [array|Record $map = null], [bool $exportValues = true], [string $resourceName = null])

This is the method to get a Record from the database.

If you want to select more records, call getIterator. This function returns null if nothing has been found. If you want it to throw an exception (for chaining) you can call get(), which is mostly a wrapper for find().
