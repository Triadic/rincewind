Class Details



[line 22]

Sometimes an associative array is simply not enough to create a query.

The DaoAttributeAssignment helps adding a few functions.  When you select records with a Dao, you can either pass an associative array containing the value assignments or a list of DaoAttributeAssignments.  It is also possible to mix both (if the value is a DaoAttributeAssignment it is used as such)  An associative array is the same as an array of DaoAttributeAssignments where the operator is always =
