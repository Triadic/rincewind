DaoResultIterator

Description
Description | Descendents | Vars (details) | Methods (details)

Implements interfaces:

  • Iterator (internal interface)

The Dao Result iterator is returned whenever a query returns more than one row.

It implements the default php Iterator Interface, so foreach() and stuff works on it.

Typical usage:

  1.  <?php
  2.  $users = $userDao->getAll(); // Returns the iterator
  3.  echo $users->count() . ' users returned.';
  4.  foreach ($users as $user) {
  5.      // ...do stuff...
  6.  }
  7.  ?>
