DaoColumnAssignment

Description
Description | Vars (details) | Methods (details)

This is a column assignment.

When you select rows with a Dao, you can either pass an associative array containing the value assignments or a list of DaoColumnAssignments. An associative array is the same as an array of DaoColumnAssignments where the operator is always =
