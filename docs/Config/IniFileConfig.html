constructor __construct [line 34]

IniFileConfig __construct( string $configFileUri, [string $defaultConfigFileUri = null])

Parses the config file. If a defaultConfigFile is provided, they get merged, and the config file overwrites the default settings.

All variables do not have to exist in the defaultConfig file, but if a section in the defaultConfigFile does not exist, then the section is ignored in the config file.
