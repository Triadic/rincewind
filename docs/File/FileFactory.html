FileFactory

Description
Description | Descendents | Methods (details)

This factory is used to get files.

The 3 common types of getting files is from a form upload, a local file or an URL. The factory makes sure that all File relevant info is set in the file object. Typical usage:

  1.  <?php
  2.    $file1 = FileFactory::get('test.txt', File::SOURCE_LOCAL);
  3.    $file2 = FileFactory::get('http://www.google.com/', File::SOURCE_HTTP);
  4.  ?>

NOTE: If you extend this class, you *have to* define getFile() and processFile() due to the nature of static::
