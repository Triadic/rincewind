Log

Description
Description | Vars (details) | Methods (details) Constants (details)

The Log class is abstract and only used with static function.

It's used to store loggers, and select the correct one for a specific context.

Example:

  1.  <?php
  2.    Log::error('Some message', 'Dao');
  3.  ?>

If you didn't set a logger for a specific context, it will be ignored.

If you don't pass a context, the context Log::GENERAL is used.
